# 魔法大脑 数据来源与合规说明

> 本文档说明魔法大脑项目的数据类型、来源、授权状态、隐私保护、行业风险提示与使用边界，满足 GOAI 赛道二（无界应用）评审"安全、合规与可追溯"维度要求。

## 1. 数据类型与来源

| 数据类型 | 来源 | 授权状态 | 说明 |
|---------|------|---------|------|
| **本地交通数据集** | 手动从 12306 官方平台、航司公开页面采集核实 | 公开信息，用于个人规划参考 | `data/routes/*.json`、`data/baselines/*.json`，仅含公开票价/时刻/站点，无个人数据 |
| **联网实时票价** | GLM-4.7 内置 `web_search` 工具联网查询公开网页 | 公开网页信息 | Agent 工具调用阶段实时查询，结果含来源网页标题与链接，可追溯 |
| **城市经纬度** | 公开地理数据（维基百科/公开 GIS） | 公开数据 | `data/geo.json`，用于地球路径绘制，无敏感信息 |
| **用户输入** | 用户在前端输入起终点/自然语言查询 | 用户主动提供 | 仅当次查询使用，不存储、不上传第三方、不建立用户画像 |

## 2. 隐私保护

- **无用户账号系统**：项目为单次查询工具，不收集用户身份信息。
- **无数据持久化**：用户查询不写入数据库，仅内存中处理当次请求，请求结束即释放。
- **无第三方追踪**：前端不嵌入任何分析/追踪 SDK（无 Google Analytics、无埋点）。
- **API Key 保护**：GLM API Key 仅存于本地 `.env` 文件（已加入 `.gitignore`），不上传仓库，代码通过环境变量读取。

## 3. 数据脱敏

本项目不涉及个人敏感信息、商业秘密或非公开业务数据，无需脱敏处理。所有数据均为公开交通信息或用户主动输入的当次查询。

## 4. 行业边界与风险提示

### 4.1 不替代专业机构决策
- 魔法大脑定位为**出行规划辅助工具**，提供中转省钱路线建议。
- **不提供订票服务**：项目无支付、无库存、无出票能力，所有"去订票"入口仅跳转外部平台。
- **不替代 OTA/12306 的最终票价**：联网查询票价为参考，实际票价以购票平台实时显示为准。

### 4.2 模型幻觉与不确定性提示
- 联网查询结果由 GLM-4.7 生成，可能存在价格偏差或信息滞后。
- 详情页底部明确标注："⚠️ 以上为联网查询参考价格，实际请以购票平台为准"。
- 对不确定结果提供置信度说明与人工确认机制（用户以购票平台为准）。

### 4.3 风险提示机制
- 联网模式结果页标注参考性质提示。
- 本地预存数据标注核实状态（✅已核实 / ⏳待核实）。
- 涉及跨境出行（如香港/澳门过关），提示用户关注签证、通关时间等实际因素。

## 5. 第三方依赖披露

| 依赖 | 类型 | 用途 | 许可证/说明 |
|------|------|------|------------|
| GLM-4.7 (智谱) | 商业大模型 API | 意图理解/规划/联网查询/结果生成 | Anthropic 兼容端点，付费 API，可替代（可换 Qwen/DeepSeek） |
| Three.js | 开源库 | 3D 地球渲染 | MIT License |
| FastAPI | 开源框架 | Web 服务 | MIT License |
| Tailwind CSS (CDN) | 开源样式库 | 前端样式 | MIT License |
| three-globe 纹理 | 公开素材 | 地球贴图 | 公开示例素材 |

### 商业 API 锁定风险说明
- GLM-4.7 为付费商业 API。本项目架构上 LLM 调用封装于 `agents/llm.py`，可替换为任意兼容 Anthropic 格式的模型（Qwen/DeepSeek 等），**迁移成本低**。
- web_search 工具为 GLM 内置能力，若迁移至无内置搜索的模型，可接 DuckDuckGo/SerpAPI 等替代，需额外开发约 50 行适配代码。

## 6. 开放与复用计划

- **代码开源**：MIT License，GitHub 公开仓库。
- **可复用组件**：
  - 多模态交通图引擎（`engine/`）——纯标准库，可独立用于任何路径规划场景
  - 多 Agent 流水线架构（`agents/`）——4 Agent 协作模式可复用于其他工具调用场景
  - 真实交通数据集（`data/`）——含核实的线路与基准，可作示例数据
- **示例数据与运行证据**：README 含快速开始、API 文档、CLI 示例输出、回归测试。

- ✅ 不替代专业机构/购票平台的最终决策
