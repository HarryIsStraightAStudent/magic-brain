# 魔法大脑 作品简介

**项目名称**：魔法大脑（Magic Brain）
**参赛赛道**：GOAI 无界应用｜Boundless Agents（出行规划细分场景）
**团队**：魔法大脑（个人参赛）

## 目标用户

预算敏感的跨城与跨境出行者——学生、年轻上班族、自由职业者等。他们注重性价比，愿意接受中转、卧铺、空铁联运等非直达走法来换取更低费用，但缺乏系统化工具去发现这些隐藏的省钱路线。

## 核心问题

主流出行产品存在明显空白：导航 App（高德/百度/Google）只做"最快路径"，OTA 平台（携程/飞猪）只卖"直达库存"，**没有任何产品做"多模态中转套利"**。真实痛点来自亲身经历——上海到香港，高铁直达 ¥966，但坐动卧到深圳（¥507）再过关到香港（¥60），全程仅 ¥567，省 41%。这种"人肉摸索省钱路线"的能力，正是 AI Agent 擅长自动化的。

## 解决方案

魔法大脑是一个 AI 交通套利搜索引擎。用户输入起点终点（或自然语言描述行程），4 个 Agent 协作完成：意图理解 → 任务规划 → 工具调用（联网查询真实票价 + 本地多目标路径搜索）→ 结果解释。输出可对比的省钱方案，配 3D 地球真实路径可视化，支持任意城市（联网模式不依赖预存数据）。

## 创新点

1. **多模态交通套利搜索**——首创将飞机、高铁、卧铺、地铁、过关整合为统一图模型，用多目标 Dijkstra 在价格/时间/换乘/舒适间优化，发现跨交通方式的中转省钱组合。
2. **真联网 Agent**——工具调用 Agent 用 GLM-4.7 内置 web_search 实时查询票价，结果含来源可追溯，非静态数据。
3. **3D 地球真实路径**——Three.js 渲染，火车贴地、飞机大圆弧，途径站点可悬停，地理直观。

## 开放/复用价值

纯 Python 标准库的交通图引擎与 4 Agent 流水线架构均可独立复用；已开源（MIT），含真实交通数据集与"如何用 GLM 构建 Agent"的范例。

## 当前进展

可运行 Demo 已完成（FastAPI + Three.js），含 5 条核实/推断线路的本地数据集，联网模式覆盖任意城市，回归测试通过。
